11一些常用终端命令

11. 一些常用终端命令：

# 新建文件夹
mkdir dir
# 删除文件夹
rm -r dir
# 新建文件，并写入“hello world”
echo "hello world" > file.text
# 使用vi编辑器打开文件,若文件不存在，就先创建后打开
vi file.text 
# 直接在终端输出文件内容
cat file.text
# 删除文件
rm file.text
#打开某个应用
start typora ./dir/path
# 查找某个目录下的某个类型文件
find ./dir -type f
