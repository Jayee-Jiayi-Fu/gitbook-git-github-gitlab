03. 文件的删除和重命名

文件删除

如需删除某个已跟踪文件，可直接在文件路径中手动删除，再把删除操作添加到暂存区。也可直接使用rm命令，一步到位：

git rm file.md

文件重命名

把文件名从"file"更改为"newFile"，同时把变更添加到暂存区：

git mv file.md newFile.md
